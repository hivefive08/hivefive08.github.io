Projects

Road Trip Planner

Developed a Python Django backend for a road trip planning application, integrating APIs from Google Maps, Amadeus (Hotels), and Yelp (Events). Implemented OAuth2 with Google for user authentication and handled route generation logic, including direct and city-based routes with hotel selection options. Designed a user-friendly frontend with JavaScript and HTML, enabling users to input trip details, view routes, and save trip data to a SQL database.

Battleship AI Agent

Created a probabilistic AI agent in Java for playing Battleship. The agent calculates the likelihood of ships being present in each cell based on the game state using the law of total probability. Considered the superposition of probabilities for each ship type, significantly improving the AI's strategic decision-making.

8 Puzzle Solver

Implemented the A* search algorithm to optimally solve the 8-puzzle problem, using Hamming and Manhattan distances as heuristics. Additionally, developed logic to detect unsolvable boards through inversion analysis, ensuring optimal performance.
